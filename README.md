MiniGame_49 - 控制台迷宫游戏
一个基于C语言开发的简易控制台迷宫游戏，展示GitHub工程化开发流程。

🎯 项目简介
MiniGame_49 是一个C语言控制台迷宫游戏，演示分支管理、协作开发、文档编写等软件工程实践。

🎮 游戏功能
控制角色 @ 在迷宫中移动探索

支持WASD键盘操作

跨平台兼容（Windows/Linux/Mac）

实时地图渲染显示

🚀 快速开始
环境要求
GCC编译器

Windows/Linux/Mac系统

编译运行

bash
cd src
gcc main.c game.c -o minigame
./minigame

操作说明

W - 向上移动
A - 向左移动  
S - 向下移动
D - 向右移动
Q - 退出游戏
