# MiniGame_49
控制台小游戏 Demo，用于展示分支管理与冲突解决

## 项目简介
简易控制台小游戏 Demo，用于展示 GitHub 协作流程（分支/PR/冲突/issue）。

## 运行方式
在项目根目录运行：
gcc src/main.c src/game.c -o minigame
./minigame

## 分支策略
- main: 稳定版本
- dev: 开发集成
- feature-ui: UI 改进
- feature-core: 核心逻辑

## 参与方式
- 提 issue 报 bug 或提建议
- fork 并发起 PR
- star/watch 支持项目
